HTML:

HTML stands for Hypertext Markup Language. It a programming language which defines the basic structure of a webpage.

HTML code is made up of:

  • Tags:

    Tags are the building blocks that contain elements that define how the information on a webpage is diplayed

  • Elements:

    Elements consits of an openning tag, content and closing tag. If a element have a opennig and a closing tag then it is a container tag but is it has only openning tag then it is a empty tag

  • Attribute:

    An Attribute is a property that provides additional information about an element. It also enhances the functionality of a tag.

CSS:

Cascading Style Sheets or css is a style sheet that contain a set of style rules for defining the layout of HTML

CSS terminology:

  • Property:

    It defines how different element will look on the webpage

  • Value:

    The option choosen for a property

  • Declaration:

    Property with value

  • Selector: Line of code indicating the target element on which property have to be applied.
  • Rule: Declaration with Selector
